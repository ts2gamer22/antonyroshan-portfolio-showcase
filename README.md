# Antony Roshan - Portfolio Website

A modern, responsive portfolio website showcasing research, projects, and professional experience in Chemical Engineering and Machine Learning.

## About

This portfolio website presents my academic journey, technical expertise, research work, and professional projects. Built with modern web technologies for a clean, professional presentation.

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **shadcn/ui** - Component library

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.
